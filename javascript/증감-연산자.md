# 증감 연산자

```javascipt
변수++ : 기존 변수 값에 1을 더한다 (후위)
++변수 : 기존 변수 값에 1을 더한다 (전위)
변수-- : 기존 변수 값에 1을 뺀다 (후위)
--변수 : 기존 변수 값에 1을 뺀다 (전위)
```

# EX 1
```javascipt
<script>
// 변수를 선언합니다.
let number = 10

// 출력합니다.
alert(++number)
alert(++number)
alert(++number)
</script>

결과 값 : 11 / 12 / 13
```

# EX 2
```javascipt
<script>
// 변수를 선언합니다.
let number = 10

// 출력합니다.
alert(number++)
alert(++number)
alert(number--)
alert(--number)
</script>

결과 값 : 10 / 12 / 12 / 10
```
