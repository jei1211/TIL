# 불 → 문자열

```javascript
String(true) → "true"
String(false) → "false"
```

# 불 → 숫자

```javascript
Number(true) → 1
Number(false) → 0

1 → 켜져있다, 존재한다
0 → 꺼져있다, 존재하지 않는다
```


# 다른 자료형 → 불

```javascript
5가지 경우 → false
0, NaN, "", null, undefined

이외의 경우 → true
```
